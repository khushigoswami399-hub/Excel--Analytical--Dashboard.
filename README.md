# Dashboard for Project

The Dashboard for this project will show the interactive dashboard in Excel, showing the process of analyzing and assessing business data via an interactive dashboard for managers to assist them in their decision-making process through access to accurate and up-to-date data.

## Overview of Project
- Clean and organize the raw data.
- Create different types of pivot tables and charts to summarize the data in different ways.
- Create an interactive dashboard that allows users to quickly generate reports or access different types of information.

## Tools
- Microsoft Excel.

## Key Highlights
- The visual representation (charts, graphs, tables) of the data allows for the identification of trends and performance.
- Reports can be produced so that managers can make decisions quicker.

## Reason for Creation
The Dashboard is part of my graduate course work (MBA) to demonstrate my analytical and reporting capabilities utilizing Excel.
